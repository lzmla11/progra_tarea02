# CoreNumeric Library

## Descripción
Implementación de funciones genéricas usando concepts de C++20.

## Funciones implementadas
- sum
- mean
- variance
- max
- transform_reduce

## Variadic
- sum_variadic
- mean_variadic
- variance_variadic
- max_variadic

## Cómo compilar
mkdir build
cd build
cmake ..
make
